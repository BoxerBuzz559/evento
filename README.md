### Run Locally

Clone the project

```bash
  git clone https://github.com/Boxerbuzz/evento.git
```

Go to the project directory

```bash
  cd evento
```

Install dependencies

```bash
  flutter pub get
```

Run the app android or ios simulator

```bash
  flutter run
```
